Novelist Local Distribution

API

Base URL: http://localhost:8080

Health: /api/v1/health

Ready: /api/v1/ready

Stats: /api/v1/stats

Generate Scene (curl)

curl -X POST http://localhost:8080/api/v1/scenes \
  -H "Content-Type: application/json" \
  -d '{
    "intention": "Hero discovers magic in an old library.",
    "chapter": 1,
    "scene": 1,
    "word_count": 800
  }'
